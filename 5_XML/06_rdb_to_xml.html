06 - Relational Database to XML Conversion

Shows how tabular data from a relational database can be represented as XML.

Key Concept: Relational databases store data in tables (rows & columns). XML represents the same data hierarchically using nested elements. Each row becomes an element, and each column becomes a child element or attribute.

Step 2: SQL Query (JOIN)

SELECT e.emp_id, e.name, d.dept_name, d.location, e.salary, e.hire_date
FROM employees e
JOIN departments d ON e.dept_id = d.dept_id
ORDER BY d.dept_name, e.name;

Step 5: PHP Code to Convert DB to XML

    
<?php
$conn = new mysqli("localhost", "root", "", "company_db");

// Query with JOIN
$sql = "SELECT e.emp_id, e.name, d.dept_name, d.location,
               e.salary, e.hire_date
        FROM employees e
        JOIN departments d ON e.dept_id = d.dept_id
        ORDER BY d.dept_name";
$result = $conn->query($sql);

// Start XML output
header('Content-Type: text/xml');
echo '<?xml version="1.0" encoding="UTF-8"?>';
echo '<employees>';

while ($row = $result->fetch_assoc()) {
    echo '  <employee id="' . $row['emp_id'] . '">';
    echo '    <name>' . $row['name'] . '</name>';
    echo '    <department>' . $row['dept_name'] . '</department>';
    echo '    <location>' . $row['location'] . '</location>';
    echo '    <salary>' . $row['salary'] . '</salary>';
    echo '    <hire-date>' . $row['hire_date'] . '</hire-date>';
    echo '  </employee>';
}

echo '</employees>';
$conn->close();
?>
