01 — Basic HTML Page Structure

Every HTML page follows a standard skeleton. Below is the minimal yet complete structure.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Example</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is where you include text and images.</p>
</body>
</html>
Key Elements:
  • <!DOCTYPE html> — Declares the document as HTML5.
  • <html> — The root element wrapping everything.
  • <head> — Contains metadata, title, and links to stylesheets.
  • <meta charset="UTF-8"> — Sets character encoding for proper text display.
  • <meta name="viewport"> — Ensures responsive design on mobile devices.
  • <title> — Sets the text shown in the browser tab.
  • <body> — Contains all visible content (text, images, links, etc.).
