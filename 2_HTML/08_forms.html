08 — HTML Forms

Forms collect user input and send it to a server for processing.

Form Shell — METHOD & ACTION

<form method="POST" action="process.php">
    <!-- form elements go here -->
</form>
  • method="POST" — Sends data in the request body (secure, no size limit).
  • method="GET" — Appends data to the URL (visible, limited to ~2048 chars).
  • action="process.php" — The server-side script that receives the form data.

Text Boxes & Text Area

<input type="text" name="fullname" placeholder="John Doe">
<input type="email" name="email">
<input type="password" name="password">
<textarea name="message" rows="4" cols="40"></textarea>

Radio Buttons

Only one option can be selected at a time (same name groups them).

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
<input type="radio" name="gender" value="other"> Other

Checkboxes

Multiple options can be selected.

<input type="checkbox" name="lang" value="html"> HTML
<input type="checkbox" name="lang" value="css"> CSS
<input type="checkbox" name="lang" value="js"> JavaScript

Drop-Down Menu (SELECT / OPTION)

<select name="country">
    <option value="" disabled selected>-- Select --</option>
    <option value="ID">Indonesia</option>
    <option value="MY">Malaysia</option>
</select>

Submit & Reset Buttons

<input type="submit" value="Submit">
<input type="reset" value="Reset">

<!-- Or using buttons -->
<button type="submit">Submit</button>
<button type="reset">Reset</button>

Complete Form Example

<form method="POST" action="process.php">
    <label>Name: <input type="text" name="name"></label>
    <label>Email: <input type="email" name="email"></label>
    <label>Gender:</label>
    <input type="radio" name="gender" value="male"> Male
    <input type="radio" name="gender" value="female"> Female
    <label>Interests:</label>
    <input type="checkbox" name="interest" value="web"> Web Dev
    <input type="checkbox" name="interest" value="ai"> AI
    <label>Country:</label>
    <select name="country">
        <option value="ID">Indonesia</option>
        <option value="MY">Malaysia</option>
    </select>
    <label>Message:</label>
    <textarea name="message" rows="3"></textarea>
    <input type="submit" value="Send">
    <input type="reset" value="Clear">
</form>
