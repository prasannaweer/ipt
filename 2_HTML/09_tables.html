09 — HTML Tables

Simple Table with BORDER

ID Name Grade
1 Alice A
2 Bob B+
3 Charlie A- 
<table border="1" cellpadding="8" cellspacing="0">
    <tr>
        <th>ID</th>
        <th>Name</th>
    </tr>
    <tr>
        <td>1</td>
        <td>Alice</td>
    </tr>
</table>

Cell Spacing & Padding

cellspacing controls the gap between cells. cellpadding controls the space inside each cell.

Featurecellspacingcellpadding
PurposeGap between cellsSpace inside cells
Default2px1px
<table border="1" cellspacing="6" cellpadding="12">
    <tr><th>A</th><th>B</th></tr>
    <tr><td>1</td><td>2</td></tr>
</table>

THEAD, TBODY, TFOOT

Item Qty Price
Pen 10 $2.50
Notebook 5 $8.00
Pencil 20 $0.50
Total $55.00 
<table>
    <thead>
        <tr><th>Item</th><th>Qty</th><th>Price</th></tr>
    </thead>
    <tbody>
        <tr><td>Pen</td><td>10</td><td>$2.50</td></tr>
    </tbody>
    <tfoot>
        <tr><td colspan="2">Total</td><td>$55.00</td></tr>
    </tfoot>
</table>

COLGROUP

<colgroup> defines column-level styles without repeating attributes on each cell.

Col 1 Col 2 Col 3 (highlighted)
A B C
D E F 
<table>
    <colgroup>
        <col span="2" style="background-color: #e3f2fd;">
        <col style="background-color: #fff9c4;">
    </colgroup>
    <tr><th>Col 1</th><th>Col 2</th><th>Col 3</th></tr>
    ...
</table>

Cell Colors with BGCOLOR

Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3
Row 3, Cell 1 Row 3, Cell 2 Row 3, Cell 3 
<tr bgcolor="#c8e6c9">
    <td>Green row</td>
</tr>

<td bgcolor="#fff9c4">Yellow cell</td>

Note: bgcolor is deprecated in HTML5. Use CSS background-color instead.
