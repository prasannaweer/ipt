03 — Style Attribute & CSS Approaches

There are three ways to apply styles in HTML: inline (style attribute), internal (style tag), and external (linked .css file).

1. Inline Styles (style attribute)

Apply styles directly to individual elements using the style attribute.

Background Color

This paragraph has a green background.

<p style="background-color: #c8e6c9;">Green background</p>

Text Color

This paragraph has red text.

This paragraph has blue text.

<p style="color: #d32f2f;">Red text</p>
<p style="color: #1565c0;">Blue text</p>

Text Alignment

Left-aligned text (default).

Center-aligned text.

Right-aligned text.

Justified text: This paragraph stretches to fill the full width of the container by adding space between words so that each line has equal length.

<p style="text-align: center;">Center text</p>
<p style="text-align: right;">Right text</p>
<p style="text-align: justify;">Justified text</p>

Font Size & Family

Large serif text.

Small monospace text.

<p style="font-size: 24px; font-family: Georgia, serif;">Serif text</p>
<p style="font-size: 12px; font-family: 'Courier New', monospace;">Mono text</p>

2. Internal CSS (style tag)

Define styles inside a <style> block in the <head>. These styles apply to the entire page.

Internal CSS Examples

This paragraph uses the .highlight class defined internally.

This paragraph is center-aligned via internal CSS.

This paragraph is right-aligned via internal CSS.

<head>
    <style>
        .highlight { background-color: #fff9c4; }
        .center-text { text-align: center; }
    </style>
</head>

3. Inline vs Internal Summary

ApproachScopeMaintenance
InlineSingle elementHard to maintain at scale
InternalEntire pageBetter for single pages
ExternalMultiple pages (shared file)Best for maintainability
