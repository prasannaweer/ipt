07 — Lists: Ordered & Unordered

Ordered Lists — <ol>

Items are numbered automatically. Use the TYPE attribute to change the numbering style.

TYPE="1" — Numeric (default)

  1. First item
  2. Second item
  3. Third item
<ol type="1">
    <li>First item</li>
    <li>Second item</li>
</ol>

TYPE="A" — Uppercase Letters

  1. Alpha
  2. Beta
  3. Gamma
<ol type="A">
    <li>Alpha</li>
    <li>Beta</li>
</ol>

TYPE="a" — Lowercase Letters

  1. apple
  2. banana
  3. cherry
<ol type="a">
    <li>apple</li>
    <li>banana</li>
</ol>

TYPE="I" — Uppercase Roman Numerals

  1. Introduction
  2. Methods
  3. Results
<ol type="I">
    <li>Introduction</li>
    <li>Methods</li>
</ol>

TYPE="i" — Lowercase Roman Numerals

  1. prologue
  2. chapter one
  3. chapter two
<ol type="i">
    <li>prologue</li>
    <li>chapter one</li>
</ol>

Start Attribute

  1. Fifth letter
  2. Sixth letter
<ol type="A" start="5">
    <li>Fifth letter</li>
</ol>

Unordered Lists — <ul>

Items are bulleted. Use the TYPE attribute (or CSS list-style-type) to change the bullet style.

TYPE="disc" — Filled circle (default)

  • HTML
  • CSS
  • JavaScript
<ul type="disc">
    <li>HTML</li>
    <li>CSS</li>
</ul>

TYPE="circle" — Open circle

  • HTML
  • CSS
  • JavaScript
<ul type="circle">
    <li>HTML</li>
    <li>CSS</li>
</ul>

TYPE="square" — Filled square

  • HTML
  • CSS
  • JavaScript
<ul type="square">
    <li>HTML</li>
    <li>CSS</li>
</ul>
