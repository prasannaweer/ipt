05 — Headings, Text Alignment & Comments

Heading Levels H1 through H6

HTML provides six levels of headings. Note: There is no H7 in the HTML standard.

H1 — Main Heading (largest)

H2 — Section Heading

H3 — Sub-section Heading

H4 — Minor Heading

H5 — Small Heading
H6 — Smallest Heading
<h1>H1 - Main Heading</h1>
<h2>H2 - Section Heading</h2>
<h3>H3 - Sub-section Heading</h3>
<h4>H4 - Minor Heading</h4>
<h5>H5 - Small Heading</h5>
<h6>H6 - Smallest Heading</h6>

Text Alignment with ALIGN Attribute

Left-Aligned Heading (default)

Center-Aligned Heading

Right-Aligned Heading

Justified Heading Text Example

<h2 align="left">Left-aligned</h2>
<h2 align="center">Center-aligned</h2>
<h2 align="right">Right-aligned</h2>
<h2 align="justify">Justified</h2>

Note: The align attribute is deprecated in HTML5. Use CSS text-align instead.
