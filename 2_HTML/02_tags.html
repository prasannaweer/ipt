02 — Text Formatting Tags

HTML provides many tags to format and structure text content.

Bold Text

<b> and <strong> make text bold.

This is bold text using <b>.

This is strong text using <strong> (semantic emphasis).

<b>bold text</b>
<strong>strong/bold text</strong>

Italic Text

<i> and <em> make text italic.

This is italic text using <i>.

This is emphasized text using <em> (semantic emphasis).

<i>italic text</i>
<em>emphasized text</em>

Underline Text

<u> underlines text.

This is underlined text.

<u>underlined text</u>

Paragraph

<p> defines a paragraph. Browsers add vertical spacing between paragraphs automatically.

This is the first paragraph.

This is the second paragraph with its own spacing.

<p>First paragraph.</p>
<p>Second paragraph.</p>

Subscript & Superscript

<sub> for subscript, <sup> for superscript.

Water is H2O and E = mc2.

H<sub>2</sub>O
E = mc<sup>2</sup>

Line Break & Horizontal Rule

<br> inserts a line break. <hr> inserts a horizontal line.

Line one.
Line two after a break.

The line above is an <hr> element.

Line one.<br>Line two.
<hr>
