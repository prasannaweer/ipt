06 — Images & Links

Images — <img> Tag

The <img> tag embeds an image. Key attributes:

With placeholder (since no real image file is available):

placeholder image (200 × 130)
<img src="photo.jpg" alt="Description" width="200" height="130">

All IMG Attributes

<img
    src="image.jpg"       <!-- Path or URL to the image -->
    alt="Alternate Text"  <!-- Text shown if image cannot load -->
    width="300"           <!-- Width in pixels -->
    height="200"          <!-- Height in pixels -->
    border="2"            <!-- Border thickness (deprecated; use CSS) -->
>

Image as a link:

<a href="https://example.com">
    <img src="logo.png" alt="Logo" width="100" height="50" border="0">
</a>

Hyperlinks — <a> Tag

Basic Link

Open Google in a new tab

<a href="https://www.google.com" target="_blank">Open Google</a>

Link to Another Page

Go to the HTML Structure page

<a href="01_structure.html">Go to Structure page</a>

Email Link

Send Email to student@mit.edu

<a href="mailto:student@mit.edu?subject=Hello&body=Message">
    Send Email
</a>

Anchor (Jump to Section)

Jump to Bottom of Page

<a href="#bottom">Jump to Bottom</a>

<div id="bottom">This is the bottom.</div>

Link Attributes

<a
    href="url"           <!-- Destination URL -->
    target="_blank"      <!-- Open in new tab/window -->
    title="Tooltip"      <!-- Tooltip on hover -->
    rel="noopener"       <!-- Security for target="_blank" -->
>Link Text</a>
↑ This is the anchor target (#bottom). Use the link above to jump here.
