04 — External CSS Example

External CSS keeps styles in a separate .css file, shared across multiple HTML pages.

Linking an External Stylesheet

<head>
    <link rel="stylesheet" href="styles.css">
</head>

Benefits

  • Single file controls the look of many pages — easy to maintain.
  • Browsers cache the CSS file, so subsequent pages load faster.
  • Cleaner HTML — no inline or internal style blocks.
